## Usuario objetivo y uso
La aplicación estará dirigida a un usuario que desee organizar su tiempo de forma fácil y rápida.

## Funcionalidades principales
 - El usuario podrá organizarse puediendo añadir tareas según prioridad, además de poder marcarlas como hechas
 - Además podrá escribir notas para tomar apuntes o guardar ideas
 - La app contará con un calendario en el que consultar las tareas futuras

## Funcionalidades opcionales
 - Sistema de rachas
 - Sistema de rutina con el que se podrá añadir rutinas diarias que el usuario podrá marcar como hechas diariamente
